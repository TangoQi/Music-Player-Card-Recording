# 汤大侠 AI 音符播放器

一个本地运行的竖版 9:16 原创音乐歌词视频生成器。

## 打开方式

直接打开：

- `outputs/cd-mv-maker.html`

或打开仓库根目录的 `index.html`，它会跳转到工具页面。

## 已有功能

- 上传音频、封面图
- 三种视频风格：音乐软件播放页、大字歌词海报、歌词卡片播放器
- 方形静止封面 / 圆形旋转封面
- 多种国风漫画场景氛围
- 歌词字体、歌词颜色、底部动画可选
- 歌词时间格编辑器，可按音频时长自动铺时间
- 画布预览独立缩放
- 浏览器内录制导出 WebM

## 录制说明

点击 `录制导出` 后，浏览器会录制当前画布和音频。录制完成后会自动下载 `.webm` 文件；如果没有自动下载，页面会显示下载链接。
