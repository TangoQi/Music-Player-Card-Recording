# 汤大侠 AI 音符播放器

这是一个竖版 9:16 原创音乐歌词视频生成器 MVP。

核心方向：

- 日系漫画感
- 国风故事感
- 手机播放器 UI
- 大字歌词海报
- 适合短视频歌词展示

## 怎么用

1. 打开 `cd-mv-maker.html`。
2. 上传音频。
3. 填写歌名和制作信息。
4. 上传封面图，或点击“生成国风封面”。
5. 粘贴歌词。
   - 推荐 LRC：`[00:12.50]这一句歌词`
   - 普通逐行歌词也可以，会按音频时长平均分配。
   - 也可以使用歌词时间格：一行一句，手动修改每句时间。
6. 选择视频风格和封面形态。
7. 点击“预览播放”。
8. 点击“录制导出”，歌曲播放完后下载 WebM 视频。

## 已实现风格

### 音乐软件播放页

适合完整歌曲、B 站、视频号、朋友圈转发。

- 上方歌曲封面
- 中间歌名和制作
- 下方当前歌词、下一句歌词
- 最下方进度条、播放按钮、音频波形

### 大字歌词海报

适合抖音、小红书、视频号短片。

- 中央大字歌词
- 左下角歌曲信息
- 小进度条和播放器控件
- 更适合爆点片段

### 歌词卡片播放器

适合温柔、文艺、故事型歌曲。

- 上方封面
- 中间精致纸张卡片
- 卡片里展示歌名、制作、歌词、进度条

## 已实现动画

- 当前歌词轻微上浮 + 淡入
- 下一句歌词小字、灰色、半透明
- 重点词微微发光
- 副歌模式下歌词会更有停留感
- 背景慢慢推进
- 光点轻微漂浮
- 封面轻微呼吸
- 圆形封面可旋转
- 进度条持续移动
- 播放按钮轻微跳动
- 音频波形上下浮动

设计原则：背景慢动，歌词准时动，进度条持续动，不让所有元素一起抢注意力。
